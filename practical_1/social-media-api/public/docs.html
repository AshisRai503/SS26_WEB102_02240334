Social Media API Documentation

This is the documentation for the Social Media API, a RESTful API for a platform.

Users

GET /api/users

Get a list of all users with pagination

Query Parameters

Parameter Type Description
page Number Page number (default: 1)
limit Number Number of users per page (default: 10)

Response

{
  "success": true,
  "count": 2,
  "page": 1,
  "total_pages": 2,
  "pagination": {
    "next": {
      "page": 2,
      "limit": 10
    }
  },
  "data": [
    {
      "id": "1",
      "username": "traveler",
      "full_name": "Karma",
      "profile_picture": "https://example.com/profiles/traveler.jpg",
      "bio": "Travel photographer",
      "created_at": "2023-01-15"
    },
    ...
  ]
}
GET /api/users/:id

Get a specific user by ID

Parameters

Parameter Type Description
id String User ID

Response

{
  "success": true,
  "data": {
    "id": "1",
    "username": "traveler",
    "full_name": "Karma",
    "profile_picture": "https://example.com/profiles/traveler.jpg",
    "bio": "Travel photographer",
    "created_at": "2023-01-15"
  }
}
POST /api/users

Create a new user

Request Body

{
  "username": "new_traveler",
  "email": "new@example.com",
  "password": "securepassword",
  "full_name": "New Traveler",
  "bio": "Adventure seeker"
}

Response

{
  "success": true,
  "data": {
    "id": "4",
    "username": "new_traveler",
    "full_name": "New Traveler",
    "email": "new@example.com",
    "profile_picture": "default-profile.jpg",
    "bio": "Adventure seeker",
    "created_at": "2023-03-20"
  }
}
PUT /api/users/:id

Update an existing user

Request Body

{
  "full_name": "Updated Name",
  "bio": "Updated bio"
}

Response

{
  "success": true,
  "data": {
    "id": "1",
    "username": "traveler",
    "full_name": "Updated Name",
    "bio": "Updated bio",
    "created_at": "2023-01-15"
  }
}
DELETE /api/users/:id

Delete a user

Response

{
  "success": true,
  "data": {}
}

Comments

GET /api/posts/:postId/comments

Get all comments for a specific post

Parameters

Parameter Type Description
postId String Post ID

Query Parameters

Parameter Type Description
page Number Page number (default: 1)
limit Number Number of comments per page (default: 10)

Response

{
  "success": true,
  "count": 5,
  "data": [
    {
      "id": "1",
      "content": "Amazing photo! ",
      "user_id": "2",
      "post_id": "1",
      "created_at": "2023-02-11",
      "user": {
        "id": "2",
        "username": "foodie",
        "full_name": "Sonam",
        "profile_picture": "https://example.com/profiles/foodie.jpg"
      }
    }
  ]
}
POST /api/posts/:postId/comments

Add a comment to a post (requires X-User-Id header)

Headers

Header Value Description
X-User-Id String User ID of the commenter

Request Body

{
  "content": "This is such a great shot! "
}

Response

{
  "success": true,
  "data": {
    "id": "42",
    "content": "This is such a great shot! ",
    "user_id": "1",
    "post_id": "1",
    "created_at": "2024-03-25"
  }
}
PUT /api/comments/:id

Update a comment (requires X-User-Id header, must be comment owner)

Request Body

{
  "content": "Updated comment text "
}

Response

{
  "success": true,
  "data": {
    "id": "42",
    "content": "Updated comment text ",
    "user_id": "1",
    "post_id": "1",
    "created_at": "2024-03-25"
  }
}
DELETE /api/comments/:id

Delete a comment (requires X-User-Id header, must be comment owner)

Response

{
  "success": true,
  "data": {}
}
